Интерфейс для датчика WT9011 BLE
Этот проект предоставляет реализацию на C++ для работы с датчиком WitMotion WT9011 DCL-BT5.0 через BLE на Windows и macOS с использованием Qt Bluetooth. Поддерживаются подключение к датчику, подписка на уведомления и парсинг данных: ускорение, гироскоп, углы, магнитометр, температура и кватернионы.
Требования

Qt 5 или новее с модулем Bluetooth
Компилятор с поддержкой C++11 или новее
BLE-адаптер на компьютере

Сборка проекта

Установите Qt и настройте среду разработки.
Создайте новый проект Qt и добавьте предоставленные файлы:
wt9011.h
wt9011.cpp
main.cpp
wt9011.pro


Откройте wt9011.pro в Qt Creator или используйте qmake:qmake wt9011.pro
make


Скомпилируйте проект для вашей платформы (Windows или macOS).

Использование

Сканирование устройств:
WT9011 sensor;
sensor.scanDevices();

Посмотрите вывод в консоли, чтобы найти MAC-адрес вашего датчика.

Подключение к устройству:Замените "00:11:22:33:44:55" на реальный MAC-адрес:
sensor.connectToDevice("00:11:22:33:44:55");


Запуск потока данных:
sensor.startStreaming();


Обработка данных:Подпишитесь на сигналы, например:
QObject::connect(&sensor, &WT9011::accelerationChanged, [](float x, float y, float z) {
    qDebug() << "Ускорение:" << x << y << z;
});


Остановка и отключение:
sensor.stopStreaming();
sensor.disconnectFromDevice();



Примечания

Убедитесь, что BLE-адаптер включен и настроен на вашем устройстве.
UUID сервиса (0000ffe0-...) и характеристики (0000ffe1-...) предполагаются стандартными для WT9011. При необходимости уточните их в документации датчика.
Код является базовой реализацией. Для специфических требований (например, других команд) адаптируйте методы sendCommand и handlePacket.
